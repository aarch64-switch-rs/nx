Struct ArrayString

#[repr(C)]
pub struct ArrayString<const S: usize> { /* private fields */ }
Expand description

Represents a C-like string of a given size (mostly like a C char[S] array)

Note that char is 4-bytes in Rust for encoding reasons, thus we must stick to u8 arrays

pub const fn from_str_truncate_null(string: &str) -> Self

Creates a ArrayString from a given &str

This creates an empty ArrayString and initializes it with the provided string. This will copy at max S - 1 bytes/chars in order to ensure that the string is NUL-terminated. This will truncate the string at the first null, so we can unconditionally return and keep it const.
